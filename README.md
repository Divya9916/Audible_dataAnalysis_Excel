# Audible-Data-Analysis-Project
Analyzed Audible audiobook data to explore ratings, reviews, pricing, and duration patterns. Performed data cleaning, transformation, and EDA to uncover insights on popular genres, top authors, customer engagement, and audiobook performance using Python and visualizations.

## Project Overview

The dataset contains audiobook information such as title, author, genre, price, rating, review count, and duration.
This project focuses on cleaning the dataset, standardizing fields, and performing Exploratory Data Analysis (EDA) entirely in Excel to identify patterns in listener engagement and audiobook performance.

## Objectives

Clean and prepare the Audible dataset

Convert unstructured duration data into usable numeric values

Analyze pricing behavior, rating trends, review patterns, and duration impact

Identify top-performing genres and authors

Create Excel-based visualizations and pivot table insights

## Skills & Techniques Used

Advanced Excel cleaning

Text extraction and transformation

Pivot Table analysis

Conditional formatting

Data categorization and grouping

Excel visualizations (bar, column, scatter, pie charts)

## Data Cleaning & Transformation

Performed complete cleaning using Excel tools:

Removed duplicates and fixed inconsistent fields

Standardized author names and genre categories

Cleaned price, rating, and review formats

Converted duration strings (e.g., “10 hrs 37 mins”) into total hours using formulas

Used Text to Columns, Flash Fill, and Find & Replace for formatting

Filled or handled missing values

## Excel Functions Used

Key Excel formulas applied in the project include:

TEXT, LEFT, RIGHT, MID – extract duration components

SUBSTITUTE, LEN, VALUE – convert strings to numeric hours

XLOOKUP – match and pull book details

COUNTIF / COUNTIFS – calculate genre or rating counts

AVERAGEIF / AVERAGEIFS – average rating per category

SUMIF / SUMIFS – aggregate reviews

IF, IFS – create labels and duration groups

Pivot Tables – analyze author, genre, duration, and rating trends

## Exploratory Data Analysis (Excel)

The analysis includes:

Rating distribution

Genre-level performance comparison

Price vs rating insights

Review count patterns

Duration group analysis

Top authors based on engagement

Identification of high-price and long-duration outliers
